ML Classification Project – Customer Churn Prediction

## Overview

This project is part of the AI Internship program. The goal of this project is to build and evaluate supervised machine learning models for customer churn prediction.

## Objective

To predict whether a customer will churn using machine learning classification algorithms.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook / VS Code

## Machine Learning Models Used

* Logistic Regression
* Random Forest Classifier
* Gradient Boosting Classifier

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis (EDA)
5. Train-Test Split
6. Cross-Validation
7. Model Training
8. Model Evaluation
9. Model Comparison

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC

## Project Outputs

* Confusion Matrix
* ROC Curve
* Feature Importance
* Model Comparison Graphs

## Files Included

* Customer_Churn_Classification.ipynb
* README.md
* Project Screenshots

## How to Run the Project

### Install Required Libraries

pip install pandas numpy matplotlib seaborn scikit-learn

### Run Notebook

Open:
Customer_Churn_Classification.ipynb

Run all cells in Jupyter Notebook or VS Code.

## Conclusion

Gradient Boosting achieved the best overall performance for customer churn prediction based on evaluation metrics and ROC-AUC score.
